## [Weave Router: Smart LLM Routing for Coding Agents](https://github.com/workweave/router) ⭐️ 7.0/10

Weave Router is an open-source model router that plugs into coding agents like Claude Code, Codex, and Cursor, using a trained RL model to intelligently route each inference request to the most cost-effective LLM, achieving 40% token cost savings without quality loss. As AI-assisted coding costs rise, especially with tokenizer changes in models like Opus 4.7, this tool addresses a critical need for cost optimization in developer workflows, potentially making AI coding assistants more accessible and sustainable. The router uses an RL model trained on tens of thousands of agent traces to decide which model to use, and it handles all necessary API translations between different providers. It is source-available under Elastic License 2.0 and can be self-hosted or used via a hosted service at weaverouter.com.
